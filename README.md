Sistema de cadastro e visualização de negociações utilizando TypeScript.
-

<strong>Funcionalidades da API</strong>

- Cadastro de transações financeiras
- Consulta de transações registradas
- Atualização de informações de uma transação existente
- Remoção de transações
- Organização das operações por regras de negócio definidas na aplicação
- Comunicação via API REST, utilizando requisições HTTP padrão
